# Slipknot ✡  - Rock Band Landing Page

## Description
This project is a professional landing page for the rock band **"Slipknot"**. It is designed to showcase the band's identity, raw music style, and live performances. The site provides fans with an immersive experience, featuring information about their **World Tour 2026**, band history, and latest discography using a strong, high-energy visual style, linear gradient for background colors in some cases etc.

## Technologies Used
The project was developed using clean, modern web standards:
* **HTML5**: For semantic structure (header, main, sections, tables, and footer).
* **CSS3**: For custom styling, layout management (Flexbox and Grid), and responsive design.
* **Bootstrap Icons**: Integrated via CDN for UI elements and social media links.

## Features
- Hero section with band promotion
- Events section with a table of upcoming concerts
- About section with band information and influences
- Multimedia section with images
- Footer with contact and social links
- Responsive design using media queries

## How to Run
1. To view the website on your local machine:
2. Download or Clone the project files.
3. Ensure index.html and style.css are located in the same folder.
4. Open the index.html file in any modern web browser (Chrome, Firefox, Safari, or Edge).
5. An active internet connection is required to load the remote images and Bootstrap icons.
